Our Philosophy for Design

1. Inclusion & Sensitivity
Designed for all age groups — children, teens, adults, elderly — and for all disability types (physical, sensory, intellectual, developmental).
Wording and content are socially acceptable, empathetic, and respectful.
Prioritizes both special needs persons (SNP) and their caregivers in usability decisions.

2. Accessibility First
Color Selection:
- High-contrast color combinations that meet WCAG 2.1 AA standards.
- Avoids overly bright or clashing tones to prevent sensory overload.
- Offers High Contrast Mode toggle for users with low vision.

Font Selection:
- Clean, legible sans-serif fonts (e.g., Open Sans, Lato).
- Comfortable default size with Font Resize Toggle (small, medium, large).
- Proper line spacing and minimal text crowding for readability.

Screen Reader Support:
- Fully navigable using Tab for keyboard users.
- Supports tap navigation for touch devices.
- Includes hover highlights to help users track their current focus.
- Screen Reader Mode Toggle — when switched on:
  • All images/icons have descriptive alt text.
  • Page structure optimized for reading order.
  • ARIA labels and roles guide assistive technology.
  • Mode can be turned on/off instantly without reloading the page.

3. Usability & Comfort
- Minimal dropdowns — most navigation options remain visible.
- Large click/tap areas and intuitive button placement.
- Consistent navigation ribbon with hover and focus states on all pages.
- Smooth, gentle fade-in/slide-in animations — non-intrusive for sensitive users.

4. Content Inclusivity
Resource Categories:
- By age group and disability type (16 possible combinations).
- Downloadable PDFs, audio files, and infographics with preview before download.
- Video library with captions, subtitles, and sign language interpretation.
- Non-stereotypical, clear category icons for quick identification.
